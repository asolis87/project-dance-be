Project; Dance

Se necesita un Saas para la gestión de academias de danza (baile). 

Requerimientos

se necesita un login con sus respectivos register, login , forgot your password
se necesita que se puedan crear, editar, eliminar y listar Alumnos
se necesita que se puedan crear, editar, eliminar y listar Docentes
se necesita que se puedan crear, editar, eliminar y listar Grupos/Clases
se necesita que se puedan inscribir o desinscribir alumnos a Grupos/Clases
se necesita manejar informacion de pagos de membresias/clases por evento
se necesita manera la asistencia de los alumnos
se necesita implementar algun tipo de recordatorio de dia de pago
se necesita tener un modulo de reportes/graficas con infromacion relevante de la academia.

Modulos

1 - Modulo Autenticacion

modulo que se encargara de la autenticacion, autorización y manejo de los usuarios del sistema.

1.1 Login

El sistema contara con un login (credenciales/google) podra iniciar con credenciales estandar osea email y contraseña y tambien podra ingresar con Google

1.2 Registro

El sistema contará con un registro de usuario , podrá registrarse de la manera estandar con email y contraseña y después tendrá que validar el correo para poder acceder de manera satisfactoria. 
También el usuario podrá registrarse con su cuenta de google para mayor comodidad.

1.3 Olvide mi contrraseña

el sistema tendra una seccion para reestablecer su contraseña ( en caso de que se haya registrado con credenciales estandar )
